$ ls research_interests

My research interests basically include:
(1) characterizing 🔍 the performance bottleneck and energy inefficiency of popular computation systems;
(2) optimizing the inefficient and nonoptimal parts inside the systems with better designs ✅ and smarter solutions 🔑 ;
(3) promoting state-of-the-art computer science and technologies and integrating with other areas, such as healthcare ❤️;

The systems I am working on are those in
  • operating systems (especially Linux kernal), containers and kubernetes;
  • cloud computing, virtualization, Java virtual machines;
  • software defined network (SDN) and network function virtualization (NFV);
  • big data systems, machine learning systems;
